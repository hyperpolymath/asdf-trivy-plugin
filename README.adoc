An {url-asdf}[asdf] plugin to install and manage {url-trivy}[Trivy] versions.

toc::[]

== Overview

=== What is Trivy?

Trivy is a comprehensive vulnerability scanner for containers, filesystems,
repositories, and configuration. It is used to identify CVEs, misconfigurations,
and secrets before deployment.

=== What is asdf?

{url-asdf}[asdf] is a universal version manager that allows you to manage
multiple runtime versions with a single CLI tool.

=== Why asdf-trivy?

[cols="1,3"]
|===
|Feature |Benefit

|**Version Switching**
|Switch between Trivy versions per project

|**Project Isolation**
|Pin Trivy per repository via `.tool-versions`

|**Reproducible CI**
|Keep CI/CD scanners aligned with local runs

|**Cross-Platform**
|Works on Linux and macOS, with Windows via WSL
|===

=== Dependencies

[source,bash]
----
# Debian/Ubuntu
sudo apt-get update
sudo apt-get install -y curl git

# Fedora/RHEL
sudo dnf install -y curl git

# macOS (via Homebrew)
brew install curl git
----

=== asdf Installation

[source,bash]
----
# Clone asdf
git clone https://github.com/asdf-vm/asdf.git ~/.asdf --branch v0.14.0

# Add to your shell (bash)
echo '. "$HOME/.asdf/asdf.sh"' >> ~/.bashrc
echo '. "$HOME/.asdf/completions/asdf.bash"' >> ~/.bashrc

# Reload your shell
exec $SHELL
----

== Installation

=== Adding the Plugin

[source,bash]
----
asdf plugin add trivy https://github.com/hyperpolymath/asdf-trivy-plugin.git
asdf plugin list
----

=== Installing Trivy Versions

[source,bash]
----
asdf list all trivy
asdf install trivy latest
asdf install trivy 0.53.0
----

=== Setting the Version

[source,bash]
----
asdf global trivy 0.53.0
asdf local trivy 0.53.0
asdf current trivy
trivy --version
----

== Usage

=== Project Configuration

[source]
----
trivy 0.53.0
----

== Configuration

This plugin uses standard asdf behavior. No additional configuration is
required.

== Troubleshooting

* Ensure `trivy` is on PATH after installation.
* Verify the installed version with `trivy --version`.
* Reinstall if a cached release is corrupted.
